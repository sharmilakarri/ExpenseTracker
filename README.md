# 💰 Expense Tracker

A simple Python-based Expense Tracker that allows users to record their expenses and calculate the total expenditure. This project is designed for beginners to understand loops, user input handling, conditional statements, and basic arithmetic operations in Python.

## 📖 Overview

Managing daily expenses is important for budgeting and financial planning. This command-line application enables users to enter multiple expenses and instantly calculates the total amount spent.

## ✨ Features

- Add multiple expenses
- Automatically calculate total expenses
- Simple command-line interface
- Exit anytime using the `quit` command
- Lightweight and beginner-friendly

## 🛠️ Built With

- Python 3

## 📂 Project Files

```
ExpenseTracker.py
```

## ▶️ Getting Started

### Prerequisites

- Python 3 installed on your system

### Running the Program

1. Clone the repository:

```bash
git clone https://github.com/your-username/expense-tracker.git
```

2. Navigate to the project directory:

```bash
cd expense-tracker
```

3. Run the application:

```bash
python ExpenseTracker.py
```
